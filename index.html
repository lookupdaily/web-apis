HTML Drag and Drop

HTML drag-and-drop uses the DOM event model and drag events, inherited from mouse events.

Uses for drag and drop:

  • File uploading
  • Reordering items in a list
  • Copying items

Full Screen API

The Fullscreen API can be used to place a given element into full-screen mode, as well as to detect when the browser enters or exits full-screen mode.

Uses for full screen:

  • Video content
  • Image carousel
  • Article or book
