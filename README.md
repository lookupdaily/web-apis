# Web APIs

A short exercise to explore the use of standard Web APIs in web development.

## Technologies used
- HTML5
- CSS3
- JavaScript

## Getting started
To view this website, simply open index.hmtl in your browser

## APIs explored

- DOM API for manipulating events on the webpage
- Audiocontext for manipulating audio or voice recorder
- geolocation
- full screen API
- web storage
- drag and drop
- fetch for making data requests
